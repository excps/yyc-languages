## DESIGN SPECIFICATIONS

### Color Scheme
**Primary Color**: Professional Blue
- Hex: #1E4D8B (Navy Blue)
- RGB: 30, 77, 139
- Represents: Trust, professionalism, stability

**Secondary Colors**: German Flag Accents
- Black: #000000 (text, headers)
- Red: #DD0000 (CTAs, emphasis)
- Gold: #FFCE00 (highlights, accents)

**Neutral Colors**:
- White: #FFFFFF (backgrounds)
- Light Gray: #F5F5F5 (sections)
- Medium Gray: #666666 (secondary text)
- Dark Gray: #333333 (primary text)

**Color Usage**:
- Primary blue for headers, navigation, footer
- Red for call-to-action buttons
- Gold for highlights and success indicators
- Neutral grays for readability

### Typography

#### Headings
**Font**: Montserrat, Raleway, or similar clean sans-serif
- H1: 42-48px (mobile: 32-36px)
- H2: 36-42px (mobile: 28-32px)
- H3: 28-32px (mobile: 24-26px)
- Weight: 600-700 (semi-bold to bold)

#### Body Text
**Font**: Open Sans, Lato, or similar readable sans-serif
- Size: 16-18px (mobile: 16px)
- Line height: 1.6-1.8
- Weight: 400 (regular)
- Color: #333333

#### Accent Text
- Testimonials: Italic, slightly larger
- Captions: Smaller, lighter weight
- Buttons: Bold, uppercase or sentence case

### Responsive Design

#### Breakpoints
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
- Large Desktop: > 1440px

#### Mobile-First Approach
- Design for mobile, enhance for desktop
- Touch-friendly buttons (min 44x44px)
- Simplified navigation on mobile
- Optimized images for smaller screens
- Fast loading on mobile networks

#### Performance Targets
- Page load: < 3 seconds
- First Contentful Paint: < 1.5 seconds
- Time to Interactive: < 3.5 seconds
- Mobile page speed score: > 85
- Desktop page speed score: > 90

### Design Elements

#### Images
**Requirements**:
- High resolution (min 1920px width for hero)
- Compressed for web (use WebP format when possible)
- Alt text for accessibility
- Lazy loading for below-the-fold images

**Image Types Needed**:
- Hero section background
- About page professional photos
- Service illustrations or photos
- Testimonial photos (with permission)
- Blog post featured images
- German cultural images

#### Icons
**Icon Library**: Font Awesome or similar
- Service icons (conversation, exam, business)
- Feature icons (flexible, certified, experienced)
- Social media icons
- Contact icons
- Navigation icons (mobile menu)

#### Visual Hierarchy
- Clear heading structure (H1 â†’ H2 â†’ H3)
- Whitespace for breathing room
- Consistent spacing (8px grid system)
- Visual separation between sections
- Focal points guide user attention

#### German Language Elements
- "Willkommen!" in hero or header
- German phrases tastefully integrated
- Cultural imagery (Brandenburg Gate, etc.)
- Flag colors in design accents
- Professional, not clichÃ©

#### WCAG 2.1 AA Compliance
- Color contrast ratio â‰¥ 4.5:1 for text
