
  # Language Tutoring Website

  This is a code bundle for Language Tutoring Website. The original project is available at https://www.figma.com/design/Adjh5pixgTtGXSTysaaHuw/Language-Tutoring-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Building the project

  Run `npm run build` to build the project for production. The output will be in the `build` directory.
